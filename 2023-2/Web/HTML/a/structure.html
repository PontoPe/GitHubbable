Estrutura Básica de um Documento HTML

Um documento HTML é composto por várias partes:

  • A declaração <!DOCTYPE html>

    • Usada para informar ao navegador que este é um documento HTML5.

  • O elemento <html>

    • O elemento raiz de um documento HTML e contém todos os outros elementos.

  • O elemento <head>

    • Contém informações meta sobre o documento, como o título da página, e não é exibido na própria página.

  • O elemento <title>

    • Usado para especificar o título do documento, que é exibido na barra de título do navegador.

  • O elemento <body>

    • Contém o conteúdo visível do documento.

  • O elemento <h1>

    • Mostra um texto como título de uma seção

  • O elemento <h2>

    • Mostra um texto como subtítulo de uma seção

  • O elemento <p>

    • Mostra um texto como parágrafo, conteúdo da seção




Tags HTML e Seus Usos

Um arquivo HTML, ou HyperText Markup Language, é a espinha dorsal de uma página da web e consiste em uma estrutura básica que define o conteúdo e a formatação de uma página da web. Geralmente, um arquivo HTML começa com uma declaração <!DOCTYPE html>, que define o tipo de documento HTML sendo utilizado. Em seguida, o elemento <html> é usado para encapsular todo o conteúdo da página. Dentro do elemento <html>, existem dois elementos principais: <head> e <body>. O elemento <head> contém informações sobre a página, como o título, metadados e referências a arquivos externos, como folhas de estilo CSS e scripts JavaScript. O elemento <body> contém o conteúdo visível da página, incluindo cabeçalhos, parágrafos, imagens, links e outros elementos visuais. Ao utilizar essa estrutura básica, os desenvolvedores podem criar páginas da web bem estruturadas e acessíveis, proporcionando uma experiência de usuário coesa e envolvente.
